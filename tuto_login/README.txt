Fonctionnement du login :

Installer les packages Flask, Flask-Login et Flask-SQLAlchemy

Pour lancer la page WEB retourner au dossier parent du dossier projet2
Faites les commandes suivantes: export FLASK_APP=projet2 ; export FLASK_DEBUG=1 ; flask run
Puis cliquer sur l'adresse HTTP présentez dans le terminal

Sur la page signup vous pouvez créer votre compte. Si l'adresse n'existe pas déjà, vous êtes redirigé sur la page login.
Si vous êtes enregistré dans la BDD vous serez envoyé sur la page profile qui vous souhaitera bienvenue, "votre nom".

Vous pouvez vous également vous déconnecter sans difficultés.
